## 项目简介

`MAA_MHXY_MG` 是一个面向《梦幻西游手游》的自动化项目，依托 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 实现界面识别、流程编排与模拟操作。

项目适合已经在 PC 模拟器环境中游玩的用户，用来减少重复点击和日常任务的手动操作成本。

> 重点说明：由于 69 级以下账号操作逻辑不同，部分功能对低等级小号的适配仍存在限制。


## 功能列表

已完成的代表性功能包括：
